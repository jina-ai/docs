Installation¶

Note

You can use Jina without even installing it via Google Colab.

Install on Mac/Linux using pip¶

To install using Python’s package manager, pip, simply run:

pip install -U jina

Read more on installing from development version, installing your own fork, cherry-picking dependencies or upgrading Jina

Install using Docker¶

docker run jinaai/jina:latest

Read more on installing from development version, supported architectures

Install on Windows¶

Jina supports WSL2 on Windows 10. Read the Jina on Windows installation guide.
